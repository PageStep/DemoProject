# DemoProject 功能

### 模块1-主页

![]https://github.com/PageStep/DemoProject/raw/master/Screenshots/Shot1.png

- 显示娱乐视频, 图片, 文字...
 - 点击视频播放, 点击图片查看大图
 - 查看大图时, 点击保存, 保存到系统相册
 - 下拉刷新, 上拉加载更多
 - 左右滑动切换视频, 图片...界面

### 模块2-登陆界面
- 显示登陆和注册界面
 - 点击弹出登陆界面或注册界面

- 推荐关注界面搭建

### 模块3-我的
- 我的界面搭建
- 设置界面搭建
 - 显示已使用缓存, 点击清除缓存
